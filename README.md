# **TDD Exchange API**

Rest API made in Nestjs, using TDD (Test Driven Development).

### Requisits to run this project

- Docker
- Docker-compose

### To run this project, follow the below steps

##### Build Project
   - `npm i`
   - `docker-compose up --build`
